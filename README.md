# ReinVim

A custom Neovim configuration for **mcreinii** that's publicly available for use.

**Version - _1.0.3_**

## Installation

```
# Clone the repo, change the location accordingly.
git clone https://github.com/ReinVim/nvim ~/.config/nvim

# Run Neovim and Install all the plugins.
nvim

# One liner.
git clone https://github.com/ReinVim/nvim ~/.config/nvim && nvim
```

## Plugin Manager

This setup uses [folke's LazyVim](https://github.com/folke/lazy.nvim) for lazy loading and easier plugin management, Thank you [folke](https://github.com/folke)!
